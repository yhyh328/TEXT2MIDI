# General MIDI (GM) Program List (0-based)

This document lists **General MIDI Level 1 program numbers** using **0-based indexing**
(as used internally in MIDI Program Change messages).

> Note:
> - Program numbers range from **0 to 127**
> - Program `0` corresponds to **Acoustic Grand Piano**
> - Channel 10 (index 9) is reserved for percussion and ignores program changes
